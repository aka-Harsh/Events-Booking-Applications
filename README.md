## ✨ Features

### 🎪 Core Features
- **Event Discovery**: Browse movies, live shows, workshops, and weekend plans with advanced filtering
- **Dynamic Pricing**: Automatic price adjustments based on booking percentage (50% sold = +10%, 80% sold = +20%)
- **QR Code System**: Generate and scan QR codes for ticket verification at venues
- **Smart Recommendations**: ML-powered event suggestions using TF-IDF and cosine similarity
- **Real-time Updates**: Live ticket availability and pricing updates

### 🎨 User Experience
- **Modern UI/UX**: Glassmorphism design with smooth animations and particle backgrounds
- **3D Animations**: Interactive cards, floating elements, and micro-interactions
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dark/Light Mode**: Automatic theme switching with user preferences
- **Progressive Web App**: Offline-capable with smooth performance

### 👥 User Management
- **Dual Authentication**: Simple password-based login for Users and Admins
- **Role-Based Access**: Separate interfaces and permissions for different user types
- **User Dashboard**: Booking history, spending analytics, and personalized recommendations
- **Profile Management**: User preferences and booking statistics

### 📊 Admin Features
- **Comprehensive Dashboard**: Revenue analytics, user statistics, and booking insights
- **Event Management**: Full CRUD operations for events with image management
- **QR Code Scanner**: Verify tickets and manage entry at venues
- **Advanced Analytics**: Sales forecasting, popular events tracking, and user behavior analysis
- **Review Management**: Monitor and moderate user reviews and ratings

## 📋 Prerequisites
- **Java 17** or higher
- **Node.js 16** or higher
- **Maven 3.6** or higher
- **Git** for version control
- **VS Code** (recommended) or any modern IDE

## 🚀 Deployment 

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/event-booking-system.git
cd event-booking-system
```
### 2. Backend Setup (Spring Boot)
```bash
# Navigate to backend directory
cd event-booking-backend

# Install dependencies and run
mvn clean install
mvn spring-boot:run
```
The backend will start on http://localhost:8080


### 3. Frontend Setup (React + Vite)
```bash
# Navigate to frontend directory (in a new terminal)
cd event-booking-frontend

# Install dependencies
npm install

# Start development server
npm run dev
```
The frontend will start on http://localhost:5173

## 🎯 Usage Guide

### Customer Interface

#### Browse Events:
- Filter by type (Movies, Live Shows, Workshops, Weekend Plans)
- Search by name, location, or tags
- View trending and popular events


#### Event Details:

- View comprehensive event information
- Check real-time pricing and availability
- Read reviews and ratings from other users

#### Booking Process:

- Select number of tickets
- See dynamic pricing updates
- Complete booking with confirmation
- Receive QR code for entry

### Dashboard Features:

- View booking history and status
- Track spending and statistics
- Get personalized event recommendations
- Manage profile and preferences



### Admin Interface

#### Event Management:

- Create, update, and delete events
- Upload images and set pricing
- Monitor ticket sales and revenue


#### Analytics Dashboard:

- View real-time booking statistics
- Analyze revenue trends and forecasts
- Track user engagement and popular events


#### QR Code Verification:

- Scan customer QR codes at venue entry
- Verify ticket authenticity
- Update booking status to completed

#### User Management:

- View user statistics and activity
- Monitor review and rating patterns
- Generate comprehensive reports
